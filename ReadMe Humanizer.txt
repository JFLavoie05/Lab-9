Le package Nugget Humanizer est utile pour personnaliser du texte comme nous le souhaitons, 
il existe plusieurs languages disponibles comme le français ou l'anglais, il comprend une multitude de fonctions 
et de méthode qui permet de faire des restrictions comme enlever des mots d'un string et de le limiter à un nombre
spécifique de caractère. Nous appelons ces fonctions comme nous appelons des méthodes de classes.
